## Authentication

All API endpoints require an API key in the `Authorization` header.

<CodeGroup>

```bash curl
curl -X GET https://api.relayer.fi/v1/metadata/types \
  -H "Authorization: ApiKey your_api_key_here"
```

```javascript
fetch('https://api.relayer.fi/v1/metadata/types', {
  headers: { 'Authorization': 'ApiKey your_api_key_here' }
})
```

```python
import requests

headers = { 'Authorization': 'ApiKey your_api_key_here' }
requests.get('https://api.relayer.fi/v1/metadata/types', headers=headers)
```

</CodeGroup>

## Base URL

All requests are made to:

```
https://api.relayer.fi/v1
```
